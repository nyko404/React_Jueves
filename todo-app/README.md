# Todo App

Aplicación de gestión de tareas construida con React, Vite y react-router-dom.

## Funcionalidades

- **Navegación**: Rutas para Inicio (`/`), Tareas (`/todos`) y Registro (`/registro`).
- **Listado de Tareas**: Obtención de datos desde JSONPlaceholder API.
- **Crear Tarea**: Formulario con validación básica para agregar tareas a la lista local.
- **Marcar Completado**: Alternar el estado de completado/pendiente de una tarea.
- **Eliminar Tarea**: Remover tareas de la lista.
- **Estados de Carga**: Indicador visual mientras se obtienen los datos.
- **Manejo de Errores**: Mensajes de error amigables si falla la API.

## Tecnologías

- React
- React Router DOM
- Fetch API
- Vite

## Cómo Ejecutar

1.  Instalar dependencias: `npm install`
2.  Iniciar servidor de desarrollo: `npm run dev`
